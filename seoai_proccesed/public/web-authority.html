How to use our tool
Web Authority

The authority of a site is an indicator key to its relevance and credibility, influencing their ability to classify into search engines like Google.

Web Authority is a powerful resource designed to help improve the authority and credibility of your website compared to others. Below you will find a guide step by step on how to use this tool effectively
1
Login
Go to SEOAI.com and enter
your username and password
by clicking the 'Log In' button.
2
Access the Reports Section
Once logged in, go to the menu on the left side of the screen and Click on the "Web Authority" option.
3
Enter URL for Analysis
In the reports section, paste the URL of the website you want analyze in the field corresponding and click on the "Analyze" button
4
Results
SEOAI.com provides various types of detailed reports to evaluate different aspects of your website and optimize it. Below the we will describe

Types of REPORTS

Site Overview Report
This report provides an overview of the status of website authority, including:
  • Domain Authority: It is a level that shows the relevance of the website. The highest levels reflect that the domain has many backlinks from other sites web, which is a web reference. It is measured within a range (0-100) with 100 being the highest level authority highest and 0 the lowest authority.
  • Page Authority: Shows the authority that has the URL provided. This number is a level importance of the page and is usually lower than the domain authority. It is measured within a range (0-100) with 100 being a page with a higher authority highest and 0 the lowest authority.
  • Trusted domain: By means of a meter exclusive ranking carried out by SEOAI, which We calculate the confidence level and the value of the possible backlinks that we can receive and/or make to main website. It is measured within a range (0- 100) with 100 being a very trustworthy website and 0 being a site unreliable website.
  • Trusted URL: It is a ranking meter exclusive made by SEOAI, which calculates the level of trust and the value of possible backlinks that we can receive and/or do to the URL. It is measured within a range (0-100) with 100 being a trusted URL and 0 an unreliable URL.
Backlinks Report

This section shows and analyzes the Backlinks that the domain and URL with the following statistics:

  • Total backlinks (all-time history): It is a metric that tracks all the backlinks that had your website since its inception. Lost backlinks (last 4 months): Metric that tracks all backlinks lost in the last 4 months
  • Link type distribution: Indicates the distribution of backlinks by link type (Text or image).
  • Dofollow Link Ratio: Metric that indicates the dofollow link ratio. It is measured within a range (0-100); 100 indicates that the relationship between links is more trusted and 0 indicates that the relationship between the links is less trustworthy.
  • Relación de enlaces activos: It is a metric that indicates the relationship of active links. It is measured within a range of (0-100), 100 indicates that the relationship between links is more trusted and 0 indicates that the relationship between the links is less trustworthy.
  • Referring domains: Shows the number of domains taken as reference.
  • Average links per domain: Shows the number of domains taken as reference.
  • Upper reference domain Calculate the number of domains taken as a superior reference
  • Anchor type distribution Shows the distribution of anchor types.
  • Domain Authority (DA): It is a metric developed by Moz that measures the probability that a complete domain appears in the top positions of search engines. It is another metric broad that considers the authority of all pages of a website as a whole. It is measured within a range (1-100) with 100 being the highest probability that search engines show that website.
  • Page Authority (PA): The authority of a page (PA) is a metric developed by Moz that measures the probability that a URL appears in the search engines. It is a broader metric than considers the authority of all the pages of a site website as a whole. It is measured within a range (1-100) 100 being the highest probability that search engines show that web page.
Competitors

This section shows data about your site's performance website and the most important competitors in the web positioning, analyzing aspects such as:

  • Backlinks: The number of backlinks this has domain from other websites.
  • Referring domains: They are the domains to which you refer to this domain with a link backlink. The higher this value, the more important the website will be.
  • Subnets: Number of different subnets that link to this domain. This evaluates that the links come from several hostings and not from the same one.
  • IPs: They are the IPs where the websites that link are hosted to the domain. Like subnets, this value indicates if the websites are hosted on a server or not.
