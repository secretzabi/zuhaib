How to Use Our Web Reports Tool?

We explain step-by-step how to make the most
of our tool and take your website
to the next level with SEO AI.

1
Login In
Go to SEOAI.com and enter
your username and password
by clicking the 'Log In' button.
2
Access the Reports Section
Once logged in, go to the menu on left side of the screen and click on the the 'Reports' option.
3
Enter the URL for Analysis
In the reports section, paste the URL of the website you want to analyze in the corresponding field and click
the 'Analyze' button.
4
Choose Report Type
SEOAI.com provides various types of detailed reports to evaluate different aspects of your website and optimize it. Below, we will
describe them.

Types of Reports

Site Overview Report
This report analyzes key SEO elements of your website, including:
Critical Errors: Issues that affect the performance and security of the site.
Medium Errors: Important problems that need attention.
Minor Errors: Small issues that can be improved
Pass Tests: Areas that are correctly configured.
SEO Report

This report analyzes key SEO elements of your website, including:

  • Title and Meta Description: Evaluation of meta tag elements.
  • Headings: Analysis of the header structure (H1, H2, etc.).
  • Content Keywords: Review of keyword optimization.
  • Image Keywords: Evaluation of alt tags in images.
  • SEO-Friendly URLs: Analysis of URL structure
  • Robots.txt: Review of the robots.txt file.
  • Page Links: Analysis of internal and external links.
  • Page Language: Detection of the language used.
  • Favicon: Verification of the presence of a favicon.

Additional Plus: If you want to improve any aspect, you can click on "Improve" to receive AI-generated optimized suggestions.

Performance Report

This report analyzes the website's performance, covering aspects such as:

  • HTML File Size: Evaluation of the HTML file size.
  • Load Time: Measurement of the page load time.
  • Page Compression: Verification of page compression
  • HTTP Requests: Number of external resources requested.
  • Image Formats: Analysis of the image formats used.
  • JavaScript Defer: Evaluation of the use of the defer attribute in scripts.
  • DOM Size: Number of nodes in the HTML page.
  • Speed Test: Specific analysis of page speed.
  • Content Size: Evaluation of the page content size.
Security Report

This report provides an analysis of the website's security, including:

  • HTTPS Encryption: Verification of HTTPS encryption.
  • HTTP/2: Evaluation of the use of the HTTP/2 protocol.
  • Mixed Content: Detection of mixed content.
  • Server Signature: Review of the server signature.
  • Insecure Cross-Origin Links: Detection of insecure cross-origin links.
  • HSTS: Evaluation of the HTTP Strict Transport Security policy.
  • Plain Text Email: Verification of plain text emails.
Miscellaneous Report

This report analyzes additional elements such as:

  • Structured Data: Evaluation of structured data tags.
  • Meta Viewport: Verification of the meta viewport tag.
  • Character Set: Evaluation of character encoding.
  • Sitemap: Review of the presence of a sitemap.
  • Associated Social Media: Detection of links to social media accounts.
  • Content Length: Evaluation of word count.
  • Text/HTML Ratio: Evaluation of the proportion of text to HTML code.
  • Inline CSS: Detection of inline CSS.
  • Deprecated HTML: Verification of deprecated HTML tags.

Additional Plus: If you want to improve any aspect, you can click on "Improve" to receive AI-generated optimized suggestions.
